# Lista de Productos de Tienda

Este proyecto es una aplicación de lista de productos que permite visualizar un catálogo, agregar nuevos ítems y eliminarlos. 
## Requisitos
- Node.js instalado.

## Instrucciones de Instalación

1. Clona o descarga este repositorio.
2. Abre la terminal en la carpeta del proyecto.
3. Instala las dependencias ejecutando:
   ```bash
   npm install


  ## Cómo ejecutar el proyecto
### Para iniciar el servidor de desarrollo, ejecuta:

Bash
npm run dev
# O si usas Create React App: npm start
Abre tu navegador en la dirección que muestre la terminal (usualmente http://localhost:5173 o http://localhost:3000).
